# Common Bundle for all traits needed

Bundle comprenant tous les traits basiques dont nous pourrions avoir besoin


## Install

```
    "repositories": [
        {
            "type": "git",
            "url": "blackajte@bitbucket.org:blackajte/traits-bundle.git"
        }
    ],
    ............
        "blackajte/traits-bundle": "^1.0"
```
